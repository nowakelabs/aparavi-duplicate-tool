How It Works

Prerequisite: You must have the Aparavi Data Suite installed and configured. This tool interacts with your Aparavi environment to identify and manage duplicate files.

  1. Download or clone the tool from GitHub.
  2. Ensure your Aparavi Data Suite is running and accessible.
  3. Run DuplicateTool.ps1 to scan your target directory for duplicate files via the Aparavi API.
  4. Review the generated CSV report for duplicates.
  5. Optionally, use DuplicateTool-Delete.ps1 to safely remove selected duplicates.
